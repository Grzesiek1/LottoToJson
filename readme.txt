Skrypt analizuje strony www w celu sparsowania i wyciągnięcia wyników gier losowych. Uzyskane wyniki zapisuje do plików json.
Aby skrypt był gotowy do działania wymagana wcześniejsza konfiguracja composera (dokładnie wykonania polecenia: "composer update" w katalogu ExternalFiles z poziomu CMD).

Istnieją alternatywne rozwiązania do pobrania niektórych z wyników:
Do pobierania niektórych wyników lotto udostępnia API
http://serwis.mobilotto.pl/mapi_v6/index.php?json=getGames
